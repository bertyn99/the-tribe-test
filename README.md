# Test de recrutement pour thetribe.io

Dans le cadre de ce test vous serez amené à réaliser un mini blog de 2 pages, la première étant la page d'accueil où seront référencer 
quelques articles et la deuxième sera une page dynamique sur laquelle s'affichera l'article.
Vous devrez réaliser seul ce test et vous n'avez le droit que d'utiliser express qui est un framework web backend en nodejs.

Il n'y a pas d'autre consigne laisser libre recours à votre imagination ;)

## c'est partit 💻🎉


### Installation

```
Vous devez cloner le projet sur votre ordinateur
```
```
Vous devez lancer l'exécutable nommer "setup" à la racine du projet, cette exécutable vous permettront d'installer toutes les dépendances requises au bon déroulement de votre test
```
